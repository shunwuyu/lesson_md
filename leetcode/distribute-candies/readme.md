偶数长度的数组， 不同的数字代表不同种类的糖果， 
一个糖果， 平均分给一个弟弟和一个妹妹。 
妹妹可以获得的最大糖果的种类数。

- 两个人糖果数量一致就可以。 
- 如果糖果种类大于n/2, 妹妹最多可以获得的糖果种类应该是n/2
- 如果小于n/2  糖果的种类数
- python list 列表
