  插入排序的工作原理是选择当前索引 i 处的元素，并从右向左搜索放置项目的正确位置。
  插入排序是一种非常简单的算法，最适合大部分已经被排好序的数据   
  算法的基本思想是一次选择一个元素，然后搜索并插入到正确的位置。
  数组被分为两个部分 —— 已排序部分和未排序的元素


  i 和 j。 i 用来跟踪外循环并表示正在排序的当前元素
  我们将每个元素向后移动一个位置，以便为要排序的新元素腾出空间。
