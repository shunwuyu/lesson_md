面试时候，除了考察常规的前端技能、框架的特性外，经常会出一些很简单的算法题目.言简意赅，在很短的时间就可以完成,但却能考察出工程师的基本计算机功力，以及知识储备。特别对于 Javascript 而言，还可以避免技术栈不同对于工程师能力的误判。

- 个数  }{   
  不可以的， 
- 正则  
  邻接的匹配的括号对，然后将其替换为空，直到不能替换为止。

- 巡检字符串，将括号分类，一类是左括号、一类是右括号。
  左括号看作是入栈信号，右括号是出栈信号。
  当出栈时，如果栈内没有与之匹配的元素，则宣告不匹配。
  当巡检完毕，如果得到空栈，则匹配，否则不匹配。
  数组实现一个简单的栈
