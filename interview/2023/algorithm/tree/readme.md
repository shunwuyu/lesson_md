- 何为递归
    函数Func(Type a,……)直接或间接调用函数本身，则该函数称为递归函数

    “递归”就意味着“反复” 反复调用自己

- 树的递归定义
    - 它可以没有根结点，作为一棵空树存在
    - 如果它不是空树，那么必须由根结点、左子树和右子树组成，且左右子树都是二叉树。

- “先”、“中”、“后”其实就是指根结点的遍历时机
