## Array
- Array.from()
  从类数组对象中创建一个新的数组
- Array.of()
根据一组参数来创建新的数组，支持任意的参数数量和类型


- find() 
 返回数组中 满足提供的 回调函数 的第一个元素的值
- findIndex()
  的索引
- every() 
  方法测试数组的所有元素是否都通过了回调函数的测试
- some() 
  方法测试是否至少有一个元素通过由提供的函数实现的测试
- sort()
  compareFunction(a, b) 小于 0 ，那么 a 会被排列到 b 之前
  a - b 3 - 4 
  compareFunction(a, b) 大于 0 ， b 会被排列到 a 之前。
  b - a
