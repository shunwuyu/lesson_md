## css
- :first-child 父元素的首个子元素
- :last-child 父元素的最后一个子元素
- clip-path: 
  裁剪路径
  元素通过定义的剪切路径
  显示裁剪过后的区域
  函数
  inset：
  inset()可以传入5个参数，分别对应top,right,bottom,left的裁剪位置,round radius（可选，圆角）
  cicle:
  1. 圆的半径，默认元素宽高中短的那个为直径，支持百分比
  2. 圆心位置，默认为元素中心点
  ellipse() : 定义一个椭圆 。
  ellipse()可以传人3个可选参数；
  1. 椭圆的X轴半径，默认是宽度的一半，支持百分比
  2. 椭圆的Y轴半径，默认是高度的一半，支持百分比
  3. 椭圆中心位置，默认是元素的中心点
  polygon() : 定义一个多边形 
  多边形的坐标（X,Y），也就是连接点
