## SVG

## box-shadow
x y 阴影模糊半径 阴影扩展半径 阴影的颜色

## filter
SVG 使用 filter 定义滤镜

## 滤镜元素

feTurbulence： 该滤镜利用Perlin噪声函数创建了一个图像。它实现了人造纹理比如说云纹、大理石纹的合成。
噪声函数：生成更自然的随机数，它常用在模拟自然物体的地方，比如地形，海水等
baseFrequency numOctaves 噪声函数需要的参数
