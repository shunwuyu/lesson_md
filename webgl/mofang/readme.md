## 数学基础

法向量
垂直平面直线所表示的向量为该平面的法向量。
向量点乘
向量大小与向量夹角cos值的积。
a = (x1, y1);
b = (x2, y2);
a*b = x1*x2 + y1*y2 = |a|*|b|*cos(θ)
θ = arccos(a*b/|a|*|b|);

转置矩阵

逆矩阵
n 阶方阵A    n 阶方阵B
