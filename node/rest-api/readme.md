REST 是浏览器与服务器通信方式的一种设计风格。

表现层状态转换
1. 互联网上所有可以访问的内容，都是资源。
2. 服务器保存资源，客户端请求资源。
3. 同一个资源，有多种表现形式。
4. 协议本身不带有状态，通信时客户端必须通过参数，表示请求不同状态的资源。
5. 状态转换通过HTTP动词表示。

URL 是资源的唯一识别符。
