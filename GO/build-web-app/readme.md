GO 是一种新的语言， 一种并发的、带垃圾回收的、快速编译的语言， 代替c/c++
- 用几秒时间编译一个大型的GO程序
- 依赖分析更容易
- GO 是静态型的语言，类型系统没有层级 更新量级
- 垃圾回收的语言， 并发执行与通信提供了基本的支持
- 多核机器上系统软件

解释型语言的游刃有余、动态类型的开发效率、静态类型的安全性。
