Skeleton Screen  骨架屏

- 完全靠手写HTML和CSS方式给每个页面定制一套骨架屏
- 利用预渲染的方式生成静态骨架屏

- 特殊的class作为获取节点信息的标记skeleton-rect和skeleton-radius

- 返回一个SelectorQuery对象实例。可以在这个实例上使用select等方法选择节点，并使用boundingClientRect等方法选择需要查询的信息。
