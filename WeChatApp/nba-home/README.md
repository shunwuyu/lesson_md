## 常犯错误1：拒绝多余的状态
思考 该变量 能不能由当前的状态推导出来
可以 就该主动推导出来
不可以 新起一个状态
删掉 light 相关的代码。


## 常犯错误2: 拒绝不明确的命名
最容易的做法：加一个 形容词 在前面
比赛相关 gameResult
支付相关 payResult

## 技术总结
利用  swiper 的 current 属性 当 current 改变 控制当前滑倒哪一部分 以此来实现动画
